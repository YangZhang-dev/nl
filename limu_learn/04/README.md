## 一、数据操作

### 1. N维数组

它是机器学习和神经网络的主要数据结构

|0-d(标量)|1-d(向量)|2-d(矩阵)|3-d|4-d|5-d|
|---|---|---|---|---|---|
|一个类别|一个特征向量|一个样本-特征矩阵|RGB图片（宽*高*通道）|批量的RGB图片|一个批量视频（多了一个时间维度)|
